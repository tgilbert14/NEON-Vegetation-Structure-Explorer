How it stays honest
Every view states its method, on the chart. Snapshot, not pooled — stand metrics use each plant's latest measurement; growth is the explicit multi-bout metric, annualised between visits. The right size measure per biome — forests are sized by tree DBH; desert shrubs are too short for a breast-height diameter, so they're sized by basal stem diameter (what NEON records for ~96–99% of desert stems), and each site is auto-classified by basal-area dominance. Basal area, not biomass — above-ground biomass needs an allometric model whose error compounds, so it is deliberately omitted; directly-measured basal area is the honest stand measure. Indices, not an inventory — per-hectare values are scaled from the sampled plots (mean ± SE across plots, the sampling unit), not a wall-to-wall census. Decreases are real, kept and flagged — diameter shrinkage between visits is verified, not deleted, and lost-tag stems are split out from biological mortality.
