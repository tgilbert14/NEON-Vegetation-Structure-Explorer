How the measurements work
The explorer uses the latest supported census for a plant’s current profile and separate supported repeat visits for change through time. Later held attempts stay disclosed rather than replacing a valid record. Stand numbers describe the sampled plots, not every tree across an entire landscape. They show slow-changing structure—not annual productivity, biomass, or carbon.
